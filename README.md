A simple template for our sinatra projects

**Remember!** 
Before all, you must install gems: $ bundle install (or $ bundle update)
To see tasks: $ rake -T

Structure:
- config: config files
- help: definitions of our functions
- public: the views' resources
- spec: spec tests
- test: app testing
- views: frond-end
